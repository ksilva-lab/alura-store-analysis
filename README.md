# Análise de Desempenho das Lojas – Alura Store

Projeto de análise de dados desenvolvido em Python com o objetivo de identificar qual loja da rede Alura Store apresenta menor desempenho, auxiliando na decisão de venda de uma unidade para investimento em um novo empreendimento.

## Objetivo

Analisar dados de vendas de quatro lojas para identificar qual apresenta menor eficiência operacional considerando:

- Faturamento total

- Avaliação média dos clientes

- Frete médio

- Categorias de produtos mais vendidas

- Produtos mais vendidos

## Tecnologias utilizadas

- Python

- Pandas

- Matplotlib

- Google Colab / Jupyter Notebook

## Principais análises realizadas

- Comparação de faturamento entre lojas

- Análise da satisfação dos clientes

- Avaliação do custo médio de frete

- Identificação das categorias mais vendidas

## Principais insights

A Loja 1 apresenta o maior faturamento da rede

A Loja 4 possui o menor faturamento

As categorias móveis e eletrônicos concentram maior volume de vendas

A Loja 3 apresenta melhor avaliação média dos clientes

## Conclusão

Com base na análise dos dados, recomenda-se a venda da Loja 4, pois ela apresenta o menor faturamento entre as unidades analisadas, indicando menor desempenho comercial dentro da rede.
